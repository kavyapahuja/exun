NeuroNova

NeuroNova is a full-stack web application developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB Atlas. The project includes user authentication with sign up and login functionality, secure password hashing using bcrypt, session management, and a contact system that stores user messages in a MongoDB database. It demonstrates frontend and backend development concepts including routing, database integration, and user data management.
